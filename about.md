# ClassSend 2

Σύστημα διαχείρισης τάξης για Windows. Δάσκαλος + μαθητές, chat, παρακολούθηση οθονών, εντολές συστήματος.

## Αναφορά σφάλματος

1. Πληκτρολόγησε `--bug` (ή `--report`) στο TUI.
2. Δημιουργείται ένα zip στο `Downloads/classsend-bugreport-<timestamp>.zip`
   με όλα τα σχετικά logs.
3. Στείλε το στο **kalotrapezis@gmail.com** με σύντομη περιγραφή.

Τα logs δεν περιέχουν προσωπικά δεδομένα — μόνο τεχνικές πληροφορίες
για τη ροή των μηνυμάτων.

## Εντολές TUI

- `--about` / `--ver` / `--version` — εμφανίζει αυτό το παράθυρο
- `--bug` / `--report` — zip με logs στο Downloads για αναφορά
- `--help` (^H) — πλήρης λίστα εντολών
- `--coffee` — διάλειμμα ☕
- `--matrix` — easter egg

## CLI flags

- `teacher.exe --version` — εμφανίζει την έκδοση και κλείνει
- `classsend-agent.exe --version` — όμοιο για τον agent
